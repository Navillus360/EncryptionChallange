# EncryptionChallange
A fun project in bash where users can use the provided decryptor script or other resources to decrypt files that are hashed via a randomly selected hashing algorithm.

# Required Packages (Root Privileges)
To encrypt the files, the script uses a package called **ccrypt**. Please ensure this is installed before running the script which can be checked using **man ccrypt** or installed using **sudo apt-get install ccrypt**
